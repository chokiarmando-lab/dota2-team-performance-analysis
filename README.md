# Dota 2 Team Performance Analysis

## 📌 Project Overview
This project explores professional Dota 2 team performance using real match data from OpenDota. The goal is to analyze team consistency, win patterns, match duration trends, and hero impact on match outcomes.

---
## 🎯 Objectives
- Analyze team win rates and performance consistency
- Identify average win duration for each team
- Examine hero pick impact on winning probability
- Detect trends in team performance over time

---

## 📊 Key Questions
- Which teams are the most consistent?
- Do some teams win faster than others?
- Which heroes contribute most to team victories?
- How does performance change over time?
